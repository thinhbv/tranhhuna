flvPlayer Plugin For FCKeditor:

Add Description:

1. Upload flvPlayer folder to  fckeditor/editor/plugins 

2. Modify fckeditor/fckconfig.js Content:
    Increase the line of code:
		FCKConfig.Plugins.Add( 'flvPlayer','en,zh-cn') ;  // FLV Video Plugin
    
3. Add a value 'flvPlayer' to FCKConfig.ToolbarSets
